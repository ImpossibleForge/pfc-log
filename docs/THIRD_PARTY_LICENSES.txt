CLOUD FEATURES (s3-info, s3-fetch)
-----------------------------------

PFC-Log v3.3 uses Python's built-in urllib library for all cloud
storage access (s3-info, s3-fetch commands). No third-party SDK
is required or bundled.

Cloud access works via pre-signed HTTPS URLs — compatible with:
  AWS S3 / S3 Glacier Instant Retrieval
  Azure Blob Storage (SAS URLs)
  Cloudflare R2, Hetzner Object Storage, MinIO, Wasabi

urllib is part of the Python Standard Library (PSF License).
No additional installation required.
